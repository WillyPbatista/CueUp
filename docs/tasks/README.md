# CueUp Tasks

Esta carpeta convierte el roadmap de CueUp en una guia de trabajo. Cada archivo es una tarea pequena con objetivo, conceptos, pasos, resultado esperado y commit sugerido.

## Como usar estas tareas

1. Elige la siguiente tarea segun el roadmap.
2. Lee el objetivo y los conceptos.
3. Implementa solo lo necesario para completar esa tarea.
4. Actualiza `docs/learning-notes.md`.
5. Ejecuta `npm run quality` cuando aplique.
6. Haz un commit con el mensaje sugerido o uno equivalente.
