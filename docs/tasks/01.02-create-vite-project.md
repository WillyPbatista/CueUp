# 01.02 - Crear el proyecto Vite

## Objetivo

Inicializar la aplicacion base con React, TypeScript y Vite.

## Conceptos que aprenderas

- Que es Vite.
- Que hace `package.json`.
- Para que sirven `main.tsx` y `App.tsx`.
- Como arranca una aplicacion React.

## Pasos

1. Crear el proyecto con Vite.
2. Instalar dependencias.
3. Ejecutar `npm run dev`.
4. Verificar que la aplicacion abre en el navegador.
5. Revisar `src/main.tsx` y `src/App.tsx`.

## Resultado esperado

La app corre localmente y muestra una pantalla inicial.

## Checklist

- [x] Existe `package.json`.
- [x] Existe `vite.config.ts`.
- [x] Existe `src/main.tsx`.
- [x] Existe `src/App.tsx`.
- [x] La app puede ejecutarse con `npm run dev`.

## Nota de aprendizaje

Explicar con tus palabras la ruta desde `index.html` hasta `App.tsx`.
