# 05.01 - Configurar Supabase

## Objetivo

Conectar React con Supabase de forma segura y preparada para Auth, Database y Realtime.

## Conceptos que aprenderas

- Proyecto Supabase.
- Variables de entorno.
- Cliente Supabase.
- Auth.
- Realtime.

## Pasos

1. Crear proyecto en Supabase.
2. Crear `.env.local`.
3. Definir `VITE_SUPABASE_URL`.
4. Definir `VITE_SUPABASE_ANON_KEY`.
5. Crear `src/lib/supabase/client.ts`.
6. Probar una llamada simple.

## Resultado esperado

La app puede comunicarse con Supabase usando variables de entorno.

## Checklist

- [ ] Proyecto Supabase creado.
- [ ] Variables de entorno configuradas.
- [ ] Cliente Supabase creado.
- [ ] `.env.local` no esta commiteado.
- [ ] Existe ejemplo de lectura o prueba.

## Nota de aprendizaje

Anotar la diferencia entre anon key y service role key.
