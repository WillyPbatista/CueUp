# 09.01 - Implementar reglas de juego

## Objetivo

Agregar reglas suficientes para que una partida tenga turnos, faltas y ganador.

## Conceptos que aprenderas

- Maquina de estados.
- Turnos.
- Scratch.
- Bola 8.
- Condicion de victoria.

## Pasos

1. Definir estado de partida.
2. Definir jugador actual.
3. Cambiar turno despues de un tiro.
4. Detectar scratch.
5. Registrar bolas metidas.
6. Detectar ganador.
7. Sincronizar eventos importantes.

## Resultado esperado

La partida ya no es solo fisica: tiene reglas que determinan quien juega y quien gana.

## Checklist

- [ ] Existe estado de turnos.
- [ ] Se detecta scratch.
- [ ] Se registran bolas metidas.
- [ ] Se detecta bola 8.
- [ ] Se detecta ganador.
- [ ] Se puede reiniciar partida.

## Nota de aprendizaje

Anotar por que las reglas deben estar separadas de Phaser.

## Commit sugerido

```txt
feat(rules): add initial game rules
```
