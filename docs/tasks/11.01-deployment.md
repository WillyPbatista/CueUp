# 11.01 - Desplegar CueUp

## Objetivo

Publicar una version jugable de CueUp y probarla con dos jugadores reales.

## Conceptos que aprenderas

- Build de produccion.
- Variables de entorno.
- Vercel.
- Supabase en produccion.
- Prueba end-to-end manual.

## Pasos

1. Ejecutar `npm run build`.
2. Crear proyecto en Vercel.
3. Configurar variables de entorno.
4. Conectar repositorio.
5. Desplegar.
6. Abrir dos sesiones y probar lobby, sala y partida.

## Resultado esperado

CueUp esta disponible online y permite probar el flujo principal multiplayer.

## Checklist

- [ ] Build local exitoso.
- [ ] Proyecto Vercel creado.
- [ ] Variables configuradas.
- [ ] Deploy exitoso.
- [ ] Supabase conectado.
- [ ] Prueba con dos jugadores completada.

## Nota de aprendizaje

Anotar que diferencias encontraste entre correr local y correr en produccion.
