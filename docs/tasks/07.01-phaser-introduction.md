# 07.01 - Integrar Phaser

## Objetivo

Montar una escena Phaser dentro de React para empezar el gameplay.

## Conceptos que aprenderas

- Game loop.
- Scene.
- Preload.
- Create.
- Update.
- Canvas dentro de React.

## Pasos

1. Instalar Phaser.
2. Crear `src/game/phaser/`.
3. Crear una escena inicial.
4. Montar Phaser desde una pagina Game.
5. Destruir la instancia al desmontar el componente.
6. Dibujar una mesa basica.

## Resultado esperado

La ruta Game muestra un canvas de Phaser con una mesa inicial.

## Checklist

- [ ] Phaser instalado.
- [ ] Escena creada.
- [ ] Canvas montado en React.
- [ ] Canvas se destruye correctamente.
- [ ] Mesa basica visible.

## Nota de aprendizaje

Anotar por que React no debe renderizar cada frame del juego.

## Commit sugerido

```txt
feat(phaser): mount initial game scene
```
