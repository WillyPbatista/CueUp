# 02.03 - Crear rutas de la aplicacion

## Objetivo

Convertir CueUp en una SPA navegable con React Router.

## Conceptos que aprenderas

- Router.
- Rutas.
- Layouts.
- Paginas.
- NotFound.

## Pasos

1. Crear `src/router/`.
2. Definir rutas principales.
3. Crear paginas iniciales en `src/pages/`.
4. Crear un layout base.
5. Agregar navegacion entre Home, Lobby, Game y Settings.

## Rutas objetivo

```txt
/
/lobby
/rooms/:roomId
/game/:matchId
/settings
*
```

## Resultado esperado

La app permite navegar entre pantallas aunque todavia no tengan logica completa.
