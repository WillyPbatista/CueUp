# 02.02 - Practicar hooks de React

## Objetivo

Aprender los hooks fundamentales con ejemplos pequenos antes de usarlos en el juego.

## Conceptos que aprenderas

- `useState`.
- `useEffect`.
- `useMemo`.
- `useCallback`.
- `useRef`.
- Hooks propios.

## Pasos

1. Crear una pantalla o componente de practica.
2. Implementar un contador con `useState`.
3. Implementar un timer con `useEffect`.
4. Guardar una referencia con `useRef`.
5. Extraer un hook propio simple.

## Resultado esperado

Tener ejemplos pequenos que expliquen como se maneja estado e interaccion en React.

## Checklist

- [x] Se uso `useState`.
- [x] Se uso `useEffect`.
- [x] Se uso `useRef`.
- [x] Se creo al menos un hook propio.
- [x] Se actualizaron las notas de aprendizaje.
