# 08.01 - Crear motor inicial de billar

## Objetivo

Modelar la fisica basica de las bolas y sus interacciones.

## Conceptos que aprenderas

- Vector.
- Velocidad.
- Friccion.
- Colision.
- Rebote.
- Estado de simulacion.

## Pasos

1. Crear tipos para Ball, Cue, Rail y Pocket.
2. Implementar vectores 2D.
3. Aplicar velocidad a una bola.
4. Aplicar friccion.
5. Detectar colision con bandas.
6. Detectar colision entre bolas.
7. Detectar bola en tronera.

## Resultado esperado

Una bola puede moverse, frenar, rebotar y entrar en una tronera en una simulacion simple.

## Checklist

- [ ] Existe modelo `Ball`.
- [ ] Existe utilidad de vector.
- [ ] La bola se mueve.
- [ ] La bola pierde velocidad.
- [ ] La bola rebota en bandas.
- [ ] Se detectan troneras.

## Nota de aprendizaje

Anotar que significa separar render de simulacion.
