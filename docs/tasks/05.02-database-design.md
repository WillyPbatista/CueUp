# 05.02 - Disenar base de datos

## Objetivo

Definir las tablas necesarias para lobby, salas, partidas y eventos.

## Conceptos que aprenderas

- Tablas.
- Relaciones.
- Enums.
- RLS.
- Migraciones.
- JSONB para eventos.

## Pasos

1. Revisar `docs/database.md`.
2. Crear carpeta `supabase/`.
3. Crear migraciones iniciales.
4. Definir tablas `profiles`, `rooms`, `room_players`, `matches` y `game_events`.
5. Definir politicas RLS basicas.

## Resultado esperado

Supabase tiene un modelo inicial para soportar partidas 1v1.
