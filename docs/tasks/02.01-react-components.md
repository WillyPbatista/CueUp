# 02.01 - Crear los primeros componentes React

## Objetivo

Construir componentes reutilizables que sirvan como base para la UI de CueUp.

## Conceptos que aprenderas

- Componentes.
- Props.
- `children`.
- Composicion.
- Clases con Tailwind.

## Pasos

1. Crear `src/components/ui/`.
2. Crear `Button`.
3. Crear `Card`.
4. Crear `Input`.
5. Crear `Badge`.
6. Usar los componentes en `App.tsx` o una pagina temporal.

## Resultado esperado

La app muestra una pantalla inicial usando componentes propios, no solo HTML directo.

## Checklist

- [x] Existe `Button`.
- [x] Existe `Card`.
- [x] Existe `Input`.
- [x] Existe `Badge`.
- [x] Los componentes aceptan props utiles.
- [x] Los componentes se ven consistentes con el tema de CueUp.

## Nota de aprendizaje

Explicar que diferencia hay entre props y children.
