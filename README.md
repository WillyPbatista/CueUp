# CueUp

CueUp es una aplicacion web para llevar el marcador de partidas de billar 1 vs 1, con autenticacion, historial de partidas, estadisticas y sincronizacion en la nube mediante Supabase.

---

## Objetivos

- Aprender React desde cero.
- Aplicar Arquitectura Limpia.
- Construir una aplicacion lista para produccion.
- Documentar todo el proceso de desarrollo.

---

## Stack tecnologico

- React
- TypeScript
- Vite
- React Router
- Supabase
- Tailwind CSS
- ESLint
- Prettier
- Husky
- lint-staged

---
